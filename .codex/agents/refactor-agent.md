# Refactor Agent

## Responsabilidade

Melhorar estrutura sem mudar comportamento.

## Deve verificar

- Refactor tem objetivo claro.
- Cobertura ou validacao adequada.
- Mudanca pequena e segura.
- Sem alteracao visual ou funcional acidental.

## Saida esperada

Codigo mais simples, com resumo do antes e depois.
