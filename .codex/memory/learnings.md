# Learnings

## 2026-06-18

- O repositorio iniciou sem arquivos de app detectados.
- A primeira entrega deve ser apenas a base `.codex/`, sem Expo, dependencias ou telas.
- A direcao visual precisa ser premium, minimalista e inspirada em iOS, evitando visual infantil.
- O produto deve nascer multi-album, com progresso, faltantes e repetidas sempre calculados por album.
- O tema por pais deve personalizar acentos visuais sem substituir a base clean e premium do app.
- As referencias visuais apontam para grid claro de figurinhas, cards arredondados, navegacao simples, ilustracoes amigaveis e hierarquia forte.
- O app deve usar elementos divertidos com sofisticacao: mascote e ornamentos sao apoio, nao decoracao dominante.
- A variacao por pais deve seguir a regra 80/20 para manter consistencia de produto.
- Expo SDK 56 atualizou automaticamente o `tsconfig.json#include` ao iniciar o Metro.
- TypeScript 6 exige `ignoreDeprecations` para manter `baseUrl` sem erro de typecheck.
- O app pode subir sem Supabase configurado se o client usar placeholder e os services bloquearem auth com uma mensagem clara.
