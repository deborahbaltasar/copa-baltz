# Skill: Create Component

## Objetivo

Criar componente reutilizavel, pequeno e tipado.

## Passos

1. Definir props minimas.
2. Evitar acoplamento com services.
3. Usar tokens de tema, espacamento e raio.
4. Prever estados visuais relevantes.
5. Exportar de forma consistente com o projeto.

## Criterios de aceite

- Props tipadas.
- Sem `any` desnecessario.
- Visual alinhado ao design system.
