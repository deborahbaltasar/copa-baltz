Esta pasta e a base de conhecimento local do projeto. Ela deve ser lida e atualizada pelos agentes antes e depois de cada tarefa relevante.

## Como usar

Antes de trabalhar, leia:
- `.codex/project-brief.md`
- `.codex/rules/global-rules.md`
- `.codex/rules/code-style.md`
- `.codex/rules/react-native-rules.md`
- `.codex/rules/supabase-rules.md`
- `.codex/rules/design-rules.md`
- `.codex/memory/architecture.md`
- `.codex/memory/design-system.md`
- `.codex/memory/progress.md`
- `.codex/memory/known-issues.md`
- O arquivo da fase atual em `.codex/phases/`

Depois de trabalhar, atualize:
- `.codex/memory/progress.md`
- `.codex/memory/changelog.md`
- `.codex/memory/backlog.md`
- `.codex/memory/known-issues.md`, se houver problemas
- `.codex/memory/decisions.md`, se houver decisoes tecnicas ou visuais
- `.codex/memory/learnings.md`, se algo importante for descoberto
- `.codex/memory/design-system.md`, se houver alteracao visual
