# Skill: Analyze Design Reference

## Objetivo

Extrair padroes de referencias visuais sem copiar interface, marca ou layout proprietario.

## Passos

1. Observar espacamento, hierarquia, cards, botoes, navegacao e estados.
2. Identificar o que combina com estetica Apple/iOS clean.
3. Identificar o que deve ser evitado.
4. Atualizar `memory/design-system.md`.
5. Nao implementar telas a menos que seja pedido.

## Saida esperada

Resumo dos padroes aproveitaveis e ajustes no design system.
