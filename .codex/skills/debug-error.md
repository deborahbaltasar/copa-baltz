# Skill: Debug Error

## Objetivo

Investigar e corrigir erro com minimo impacto.

## Passos

1. Reproduzir ou localizar o erro.
2. Ler arquivos relacionados antes de editar.
3. Isolar causa provavel.
4. Corrigir de forma focada.
5. Rodar validacoes relevantes.
6. Registrar aprendizado ou issue se necessario.

## Criterios de aceite

- Causa explicada.
- Correcao validada.
- Risco residual documentado.
