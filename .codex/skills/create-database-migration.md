# Skill: Create Database Migration

## Objetivo

Criar migration Supabase/Postgres segura e reprodutivel.

## Passos

1. Ler `memory/database.md` e regras Supabase.
2. Definir schema, constraints e indexes.
3. Habilitar RLS quando houver dados sensiveis.
4. Criar policies junto da tabela quando aplicavel.
5. Validar SQL localmente ou no Supabase CLI quando existir.
6. Atualizar memoria de banco.

## Criterios de aceite

- Migration versionada.
- Constraints e indexes definidos.
- RLS e policies documentadas.
