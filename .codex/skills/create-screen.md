# Skill: Create Screen

## Objetivo

Criar uma nova tela seguindo arquitetura e design system.

## Passos

1. Ler fase atual, regras React Native e design system.
2. Definir responsabilidade da tela.
3. Identificar componentes reutilizaveis existentes.
4. Manter regra de negocio fora da tela.
5. Implementar loading, erro e vazio quando houver dados remotos.
6. Validar typecheck/lint quando existirem.
7. Atualizar memoria e changelog.

## Criterios de aceite

- Tela nao contem logica complexa.
- Usa tokens visuais existentes.
- Navegacao esta coerente com Expo Router.
