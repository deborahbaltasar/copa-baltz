# Skill: Refactor Code

## Objetivo

Melhorar estrutura sem alterar comportamento.

## Passos

1. Definir objetivo do refactor.
2. Verificar cobertura ou validacao existente.
3. Fazer mudancas pequenas.
4. Rodar validacoes.
5. Registrar decisao se alterar arquitetura.

## Criterios de aceite

- Comportamento preservado.
- Codigo ficou mais simples ou mais coeso.
